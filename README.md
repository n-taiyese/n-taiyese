## 🗂️ Portfolio Projects

### 🛍️ [Retail Sales Performance Dashboard](https://github.com/mcfapejnr007/retail-dashboard)
> Python · SQL · Power BI · Excel

Built an automated analytics layer replacing manual weekly Excel reporting for a multi-location retail operation. Analysed 15,000 POS transactions across 4 stores — uncovering that the top 3 SKUs drive ~52% of revenue and that holiday peaks run 2× baseline volume.

- ETL pipeline with documented anomaly detection (245 injected data issues resolved)
- 5 analytical SQL queries using window functions: LAG(), RANK(), rolling averages, Pareto ranking
- Star-schema outputs structured for direct Power BI import

### 📡 [Customer Churn Analysis — Telecom](https://github.com/mcfapejnr007/churn-analysis)
> Python · scikit-learn · SQL · Tableau

Identified at-risk customers and quantified revenue exposure for a telecom provider losing customers without knowing why. Built a predictive model flagging accounts before they churn.

- 44.4% overall churn rate; month-to-month contract customers churn at 57% vs 25.9% for 2-year
- Logistic Regression (AUC 0.728) + Random Forest for feature importance — interpretability first
- Cohort SQL queries isolating a high-risk segment worth ~$25K–$37K/month in recoverable revenue

### ⚙️ [Energy Operations Data Audit & Reporting Automation](https://github.com/mcfapejnr007/ops-audit)
> Python · pandas · openpyxl · Excel

Automated multi-site data consolidation and QA auditing across 5 field sites — replacing 4–6 hours of weekly manual work with a pipeline that runs in under 5 seconds.

- DAMA DMBOK-aligned audit framework: Completeness, Validity, Uniqueness, Consistency, Timeliness
- Detected 76 data quality flags (vs 35 injected — pipeline caught additional real inconsistencies)
- 4-sheet Excel output with executive summary, consolidated data, site breakdown, and colour-coded QA log
